# Kalkulator Kode Harga - Multi Platform

Aplikasi kalkulator untuk mengkonversi kode harga dengan peta huruf CINTAROSUL = 1234567890.

## Platform yang Tersedia

### 1. **React Native** (iOS & Android)
Folder: `mobile-reactnative/`

### 2. **Flutter** (iOS & Android)
Folder: `mobile-flutter/`

### 3. **Python Desktop** (Windows, Mac, Linux)
Folder: `desktop-python/`

### 4. **Web App** (React)
Folder: `web-react/`

### 5. **HTML Original**
Folder: `web-html/`

## Cara Menggunakan

Pilih platform sesuai kebutuhan Anda, lalu ikuti instruksi setup di folder masing-masing.

## Fitur Utama
- Konversi kode huruf ke angka otomatis
- Perhitungan harga dalam Rupiah
- Interface yang user-friendly
- Cross-platform compatibility

## Peta Konversi
- **CINTAROSUL = 1234567890**
  - C=1, I=2, N=3, T=4, A=5, R=6, O=7, S=8, U=9, L=0
